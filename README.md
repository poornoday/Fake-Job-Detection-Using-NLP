# Fake Job Detection System 🔍

## Overview
A machine learning & NLP-based web application that detects fraudulent job postings.

## Features
- Fake/Real job prediction
- BiLSTM + ML models
- Confidence score
- Feedback & flagging system
- SQLite database storage

## Tech Stack
- Python
- Flask
- TensorFlow
- Scikit-learn
- HTML/CSS/JS

## How to Run
1. Create virtual environment
2. Install dependencies
   pip install -r requirements.txt
3. Run database
   python init_db.py
4. Start server
   python app.py
5. Open browser
   http://127.0.0.1:5000
